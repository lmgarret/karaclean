  <action>
Create `.golangci.yml` at the repo root with the following exact content:

```yaml
version: "2"

linters:
  default: standard
  enable:
    - gocyclo
    - godot
    - misspell
    - noctx

  settings:
    gocyclo:
      min-complexity: 15
    godot:
      scope: toplevel
    errcheck:
      check-type-assertions: true
```

Key decisions per user and research:
- `version: "2"` -- golangci-lint v2 format (v1 format will not parse)
- `default: standard` enables: errcheck, govet, ineffassign, staticcheck, unused (note: `gosimple` is merged into `staticcheck` in v2, so not listed separately)
- Extras: gocyclo (complexity), godot (comment style), misspell (typos), noctx (missing context)
- `gocyclo` min-complexity: 15 (reasonable for small project)
- `godot` scope: toplevel (only check top-level comments)
- `errcheck` check-type-assertions: true (catch unchecked type assertions)
  </action>
