Function signature:
```go
func MatchesExceptions(b Bookmark, ex *config.Exceptions) bool
```

Implementation -- OR semantics, short-circuit on first true:

1. If `ex == nil`, return `false` (no exceptions = no protection).

2. Check `ex.Favourited`: if non-nil and `b.Favourited == *ex.Favourited`, return `true`.

3. Check `ex.HasTag`: if non-nil, iterate `b.Tags` looking for exact match `tag == *ex.HasTag` (case-sensitive with `==`, same as conditions). If found, return `true`.

4. Check `ex.HasNote`: if non-nil and `*ex.HasNote == true`, check `strings.TrimSpace(b.Note) != ""`. If that is true, return `true`. If `*ex.HasNote == false`, check `strings.TrimSpace(b.Note) == ""`. If that is true, return `true`.

5. Check `ex.Archived`: if non-nil and `b.Archived == *ex.Archived`, return `true`.

6. Return `false` (no exception fired).

Add a doc comment: `// MatchesExceptions returns true if the bookmark is protected by any exception clause (OR semantics). Short-circuits on first match. Returns false for nil Exceptions.`
